# AI 提示词工程学习仓库

🚀 这是一个使用 **AI 引导式学习方法** 系统化学习 Prompt Engineering（提示词工程）的仓库。

以 Claude Code 作为互动式导师，通过苏格拉底式教学、个性化反馈和系统性的进度追踪，从零开始掌握提示词工程的核心技能。

---

## 工作原理

本仓库使用 Claude Code 作为互动式提示词工程导师，它能够：
- 使用**苏格拉底式提问法**进行教学（首先询问你已知的内容，再基于你的基础构建新知识）
- 提供**简洁的解释**（约 200 词），配合实际可运行的提示词示例
- 通过**追问验证**你的理解，确保你真的掌握了，而不是"感觉懂了"
- 根据你的回答**调整教学风格**和节奏
- **追踪每次学习会话**，自动记录进度、知识差距和已掌握的主题

## 仓库结构

```
/sessions/                    # 每日学习会话记录
  /YYYY-MM-DD/               # 每个学习日一个文件夹
  SESSION-TEMPLATE.md        # 会话记录模板

/progress/                    # 学习进度的唯一数据来源
  ai-prompt-study-tracker.md # 综合追踪器，覆盖全部 41 个主题

CLAUDE.md                     # AI 导师行为指令（苏格拉底方法等）
README.md                     # 本文件
LEARNING-FRAMEWORK.md         # 通用学习框架（可复用到其他知识领域）
```

## 知识体系概览

本学习计划覆盖 **7 大模块、41 个主题**：

| 模块 | 权重 | 核心内容 |
|------|------|----------|
| **A. 提示词基础理论** | 20% | LLM 原理、提示词要素、幻觉与局限、评估方法 |

| **E. 提示词优化与调试** | 10% | 调试方法论、元提示词、自动优化 |
| **F. 提示词安全与防护** | 10% | 注入防御、越狱防护、红队测试 |
| **G. 实战场景应用** | 10% | 代码、写作、数据分析、多模态等场景 |

## 如何使用

### 每日学习会话

1. 在此仓库中打开 Claude Code：
   ```bash
   cd AI-Prompt-Study
   claude-code
   ```

2. 自然地提问——就像与一位资深提示词工程师对话：
   - "CRISPE 框架是什么？怎么用？"
   - "帮我看看这个提示词哪里可以优化"
   - "Chain of Thought 和 Tree of Thoughts 有什么区别？"

3. 回答 Claude 的理解检查问题——这是学习的关键环节

4. 每次会话后，Claude 会自动记录学习情况

### 复习课

当你想复习时，直接问 Claude：
- "让我们复习一下我之前遇到困难的主题"
- "给我出几道提示词设计的实战题"
- "展示我的学习进度，哪些模块还需要加强"
- "用我学过的技术，帮我优化这个真实场景的提示词"

### 追踪你的进度

查看 `/progress/ai-prompt-study-tracker.md` 了解整体学习状态，包括已掌握的 41 个主题、知识差距和学习计划。

## 学习理念

**引导式学习方法：**
- 对话式、无评判——没有"愚蠢的问题"
- 在你已有知识基础上构建——不会从头讲你已经懂的东西
- 前进之前检查理解程度——确保真正的掌握
- 适应你的学习风格——快的地方快，需要细讲的地方慢
- 专注深度理解——不只是记住模板，而是理解为什么这样写有效

**实践导向：**
提示词工程是"做"的学问。每个概念都配有实际可运行的示例，鼓励你立即动手修改和实验。

## 如何用本仓库开始你自己的学习

1. **克隆或复制本仓库**到你的工作目录

2. **清除示例学习历史**（全新开始）：
   ```bash
   rm -rf progress/ sessions/
   ```

3. **运行 Claude Code**：
   ```bash
   claude-code
   ```

4. **开始第一个问题**：直接问你想了解的提示词技术，Claude 会自动引导你并追踪进度。

`CLAUDE.md` 包含了导师的全部行为指令，不需要额外配置。

## 如果你想学习其他知识领域

本仓库附带一份 **通用学习框架**（`LEARNING-FRAMEWORK.md`），提炼了这套方法的核心模式。你只需要：
1. 定义你的知识模块和主题列表
2. 替换框架中的占位符
3. 即可用相同方法学习任何系统化知识

支持的场景：职业资格认证（CPA、CFA、PMP）、编程语言系统学习、计算机科学基础、产品设计方法论等。
