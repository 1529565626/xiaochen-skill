# 提示词书写课程 · 进度追踪

**最后更新**：2026-05-21
**课程状态**：进行中

## 全局课程任务清单

- [x] **任务 1：理解 AI 对话模型的基本原理**
  - 教学目标：知道 AI 如何生成文本（基于上下文预测），以及"提示词"就是引导预测的输入
  - 完成标准：学生能用自己的话说出"提示词是向 AI 传递指令和背景的方式"

- [ ] **任务 2：写好提示词的第一原则——清晰与具体**
  - 教学目标：掌握使用具体名词、动词、限制条件来消除歧义
  - 完成标准：改写后的提示词包含了主题、输出长度或风格等至少两个明确约束

- [ ] **任务 3：通过角色扮演和风格指导提升输出**
  - 教学目标：学会使用"你是……"角色设定以及指定语气/格式
  - 完成标准：学生主动使用了角色设定和至少一种风格修饰语

- [ ] **任务 4：使用示例（Few-shot）来规范输出**
  - 教学目标：理解 Few-shot 的价值，能给出 1-3 个示例来规范输出格式和内容模式
  - 完成标准：提示词中至少包含两个清晰的示例对

- [ ] **任务 5：分解复杂任务——思维链（Chain-of-Thought）**
  - 教学目标：学会引导 AI 一步步推理
  - 完成标准：提示词包含"逐步推理"、"首先，然后"或类似短语

- [ ] **任务 6：调试与迭代——如何优化不满意的回答**
  - 教学目标：掌握修正提示词的常用策略
  - 完成标准：成功将一次失败的回答通过修改提示词变好

- [ ] **任务 7：高级技巧——检索增强输出（提供参考文本）**
  - 教学目标：学会将外部文档粘贴进提示词，让 AI 基于内容回答
  - 完成标准：正确地在提示词中包含参考文本块并使用明确指令

- [ ] **任务 8：控制输出格式——结构化输出（JSON、XML、Markdown）**
  - 教学目标：要求输出可被程序解析的格式
  - 完成标准：AI 的响应直接被 JSON 解析器接受

- [ ] **任务 9：避免幻觉与安全使用技巧**
  - 教学目标：知道 AI 可能编造信息，学会要求引用来源或承认不知道
  - 完成标准：提示词中包含"如果你不知道，请直接说不知道"或类似短语

- [ ] **任务 10：综合实战——为真实任务设计一套提示词**
  - 教学目标：综合运用所有技巧，为实际场景编写提示词集合
  - 完成标准：交付一份包含至少 3 个提示词的文档，每个提示词应用了至少 2 个本课技巧
